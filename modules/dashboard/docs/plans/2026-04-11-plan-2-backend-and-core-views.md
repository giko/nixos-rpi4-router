# Router Dashboard — Plan 2: Backend + Core Views

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the full observability backend (13 collectors → 9 endpoints needed for the designer-delivered views) and the 4 designer-delivered pages (Overview, VPN Pools, Clients, AdGuard) on top of the Plan 1 foundation.

**Architecture:** A three-tier collector pattern (2 s hot / 5 s medium / 30 s cold) writes into a single `sync.RWMutex`-guarded `State` cache. HTTP handlers only read from the cache — they never exec or block on I/O. A new `topology` package reads a JSON config file generated by the NixOS module at eval time to know which tunnels, pools, static leases, and pooled rules exist. The React frontend gains shadcn/ui primitives, a TanStack-Query + React-Router shell, and the 4 designer-delivered pages rebuilt from the existing HTML mocks.

**Tech Stack:** Go 1.22+ stdlib + `golang.org/x/time/rate` + `golang.org/x/sync/singleflight`, React 18, TypeScript, Vite 5, Tailwind 3, shadcn/ui, TanStack Query v5, React Router v6, Recharts.

## Task completion protocol — `/codex:review` per task

Every task ends with a Codex review step. The flow per task is:

1. **Implementer** subagent runs all the TDD/build/test steps inside the task and commits at the end.
2. **Controller** (the main agent) runs `/codex:review --wait` on the just-made commit.
3. If Codex returns findings, the controller dispatches a fix subagent with the exact findings pasted in. The fix lands as a follow-up commit (NOT an amend — new commit).
4. The controller then runs the normal spec-compliance + code-quality review loops (`subagent-driven-development` skill defaults).
5. Only when Codex has no findings (or its findings have been addressed) AND spec + quality reviewers approve, the task is marked complete in the todo list.

**Important:**

- Use `--wait` (not `--background`) for Codex reviews so the controller can act on findings immediately.
- The implementer subagent does NOT run `/codex:review` itself — it has no access to the slash command. Only the controller runs it.
- Codex review is scoped to the commits produced in the current task, not the whole worktree.

Each task's last step is literally written as:

> - [ ] **Step N: Codex review**  
>   Controller runs `/codex:review --wait`. Address findings, then re-review.

That step is considered complete only after Codex either approves or all findings have been fixed in follow-up commits.

## Runtime topology via config file

The Nix module knows which tunnels, pools, pooled rules, and static leases exist at eval time; the Go binary needs that same information at runtime. The bridge is a JSON file written by `pkgs.writeText` and passed as `--config-file=<path>` on the dashboard's ExecStart line.

**Schema (`dashboard-config.json`):**

```json
{
  "tunnels": [
    { "name": "wg_sw", "interface": "wg_sw", "fwmark": "0x20000", "routing_table": 131072 },
    { "name": "wg_us", "interface": "wg_us", "fwmark": "0x30000", "routing_table": 196608 }
  ],
  "pools": [
    { "name": "all_vpns", "members": ["wg_sw", "wg_us", "wg_nl", "wg_uk"] }
  ],
  "pooled_rules": [
    { "sources": ["192.168.20.3", "192.168.1.125"], "pool": "all_vpns" }
  ],
  "static_leases": [
    { "mac": "1C:1A:DF:5C:D3:88", "ip": "192.168.1.194", "name": "XBOX" }
  ],
  "allowlist_enabled": true,
  "allowed_macs": [
    "44:fa:66:65:86:43"
  ],
  "lan_interface": "eth0"
}
```

`allowlist_enabled` is true only when `router.nftables.allowedMacs` is actually configured; when the option is `null` on this deployment, the flag is false and the dashboard renders every client's allowlist status as `n/a` instead of falsely flagging them as `blocked`. This matters because `router.nftables.allowedMacs` is declared `nullOr` in `modules/nftables.nix` — some installs won't use the MAC allowlist at all.

The dashboard reads this on startup, fails hard if the file is present-but-unparseable, and tolerates a missing file (empty topology → all affected endpoints return empty slices).

```go
// Package envelope wraps every JSON API response in the shape
// { "data": ..., "updated_at": ..., "stale": bool } defined by spec §7.4.
package envelope

import (
	"encoding/json"
	"net/http"
	"time"
)

// Response is the on-the-wire envelope. Exported so callers can reuse it
// for hand-built bodies (rarely needed — WriteJSON is the normal path).
type Response struct {
	Data      any        `json:"data"`
	UpdatedAt *time.Time `json:"updated_at"`
	Stale     bool       `json:"stale"`
}

// WriteJSON serialises `data` inside the standard envelope and writes it
// to w with the supplied status code. A zero `updated` renders as JSON
// null (indicating "never collected") and should be paired with stale=true.
func WriteJSON(w http.ResponseWriter, status int, data any, updated time.Time, stale bool) {
	w.Header().Set("Content-Type", "application/json; charset=utf-8")
	w.WriteHeader(status)

	var updatedPtr *time.Time
	if !updated.IsZero() {
		u := updated.UTC()
		updatedPtr = &u
	}

	// Errors from Encode only happen if the underlying writer fails, which
	// for httptest.ResponseRecorder or a real http.ResponseWriter means the
	// client hung up. There's nothing useful to do about it — the header is
	// already written — so we ignore the error. slog would double-log in
	// aggregate middleware if we surfaced it here.
	_ = json.NewEncoder(w).Encode(Response{
		Data:      data,
		UpdatedAt: updatedPtr,
		Stale:     stale,
	})
}
```

```go
	RXBytesTotal  uint64            `json:"rx_bytes_total"`
	TXBytesTotal  uint64            `json:"tx_bytes_total"`
```

```go
// Package state holds the dashboard's in-memory cache. Every endpoint
// handler reads from State and nothing else — handlers never trigger
// blocking I/O. Collectors (see internal/collector) write to State on
// their own cadence. All access is guarded by a single RWMutex because
// reads vastly outnumber writes and the section count is small enough
// that per-section mutexes would be over-engineering.
package state

import (
	"sync"
	"time"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
)

// State is the central cache. Zero value is ready to use but returns
// zero times for every section, which IsStale treats as "never collected".
type State struct {
	mu sync.RWMutex

	system        model.SystemStats
	systemUpdated time.Time

	traffic        model.Traffic
	trafficUpdated time.Time

	tunnels        []model.Tunnel
	tunnelsUpdated time.Time

	pools        []model.Pool
	poolsUpdated time.Time

	clients        []model.Client
	clientsUpdated time.Time

	adguard        model.AdguardStats
	adguardUpdated time.Time
}

// New returns an empty, ready-to-use State.
func New() *State { return &State{} }

// ---------- System ----------

func (s *State) SetSystem(v model.SystemStats) {
	s.mu.Lock()
	defer s.mu.Unlock()
	s.system = v
	s.systemUpdated = time.Now().UTC()
}

func (s *State) SnapshotSystem() (model.SystemStats, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	return s.system, s.systemUpdated
}

// ---------- Traffic ----------

func (s *State) SetTraffic(v model.Traffic) {
	s.mu.Lock()
	defer s.mu.Unlock()
	s.traffic = v
	s.trafficUpdated = time.Now().UTC()
}

func (s *State) SnapshotTraffic() (model.Traffic, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	return s.traffic, s.trafficUpdated
}

// ---------- Tunnels ----------

func (s *State) SetTunnels(v []model.Tunnel) {
	s.mu.Lock()
	defer s.mu.Unlock()
	// Defensive copy — collector may still hold the slice.
	s.tunnels = append(s.tunnels[:0], v...)
	s.tunnelsUpdated = time.Now().UTC()
}

func (s *State) SnapshotTunnels() ([]model.Tunnel, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	out := append([]model.Tunnel(nil), s.tunnels...)
	return out, s.tunnelsUpdated
}

// ---------- Pools ----------

func (s *State) SetPools(v []model.Pool) {
	s.mu.Lock()
	defer s.mu.Unlock()
	s.pools = append(s.pools[:0], v...)
	s.poolsUpdated = time.Now().UTC()
}

func (s *State) SnapshotPools() ([]model.Pool, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	out := append([]model.Pool(nil), s.pools...)
	return out, s.poolsUpdated
}

// ---------- Clients ----------

func (s *State) SetClients(v []model.Client) {
	s.mu.Lock()
	defer s.mu.Unlock()
	s.clients = append(s.clients[:0], v...)
	s.clientsUpdated = time.Now().UTC()
}

func (s *State) SnapshotClients() ([]model.Client, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	out := append([]model.Client(nil), s.clients...)
	return out, s.clientsUpdated
}

// ok=false if not found. Zero allocation — callers receive a value, not
// a pointer.
func (s *State) SnapshotClient(ip string) (model.Client, time.Time, bool) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	for _, c := range s.clients {
		if c.IP == ip {
			return c, s.clientsUpdated, true
		}
	}
	return model.Client{}, time.Time{}, false
}

	s.mu.Lock()
	defer s.mu.Unlock()
	s.adguard = v
	s.adguardUpdated = time.Now().UTC()
}

func (s *State) SnapshotAdguard() (model.AdguardStats, time.Time) {
	s.mu.RLock()
	defer s.mu.RUnlock()
	return s.adguard, s.adguardUpdated
}

// IsStale reports whether `updated` is older than 2× the collector's
// normal interval. The zero time is always stale (never-collected case).
// A value exactly at the 2× boundary is NOT stale — we use strict >.
func IsStale(updated time.Time, interval time.Duration) bool {
	if updated.IsZero() {
		return true
	}
	return time.Since(updated) > 2*interval
}
```

```go
// Package collector drives the dashboard's background data-collection
// goroutines. Each Collector runs on its own ticker governed by its Tier.
// The Runner handles scheduling, error logging, and graceful shutdown;
// collector implementations just pull data from their source and write
// it into the shared state.State.
package collector

import (
	"context"
	"log/slog"
	"sync"
	"time"
)

// Tier determines how often a Collector runs.
type Tier int

const (
	// Hot tier — 2 s. Used for metrics that change continuously (/proc/net/dev,
	// wg show dump, /proc/stat, thermal).
	Hot Tier = iota
	// Medium tier — 5 s. Used for metrics that change frequently but aren't
	// worth tighter polling (systemctl is-active, dnsmasq leases, AdGuard stats).
	Medium
	// Cold tier — 30 s. Used for expensive or slow-changing queries
	// (conntrack -L scans, nft ruleset dumps, ip rule show).
	Cold
)

// Interval returns the ticker period for this tier.
func (t Tier) Interval() time.Duration {
	switch t {
	case Hot:
		return 2 * time.Second
	case Medium:
		return 5 * time.Second
	case Cold:
		return 30 * time.Second
	}
	return 0
}

// String returns a short human-readable tier name used in log fields.
func (t Tier) String() string {
	switch t {
	case Hot:
		return "hot"
	case Medium:
		return "medium"
	case Cold:
		return "cold"
	}
	return "unknown"
}

// Collector is implemented by every data-source collector.
type Collector interface {
	// Name is a stable identifier used in structured logs.
	Name() string
	// Tier determines the cadence. Collectors can change tier between
	// process restarts but not within a single process lifetime.
	Tier() Tier
	// Run performs one collection pass. On error the Runner logs it and
	// leaves previous State in place — handlers will surface it via the
	// envelope's stale flag when updated_at falls behind.
	Run(ctx context.Context) error
}

// Runner owns a slice of Collectors and their goroutines.
type Runner struct {
	logger     *slog.Logger
	collectors []Collector
	wg         sync.WaitGroup
}

// NewRunner constructs a Runner. The slice is shallow-copied so callers
// may reuse it.
func NewRunner(logger *slog.Logger, collectors []Collector) *Runner {
	if logger == nil {
		logger = slog.Default()
	}
	cs := make([]Collector, len(collectors))
	copy(cs, collectors)
	return &Runner{logger: logger, collectors: cs}
}

// Start launches one goroutine per Collector. Start returns immediately;
// callers must eventually cancel ctx and optionally call Wait to join.
func (r *Runner) Start(ctx context.Context) {
	for _, c := range r.collectors {
		r.wg.Add(1)
		go r.loop(ctx, c)
	}
}

// Wait blocks until every collector goroutine has exited. Safe to call
// after Start even from multiple goroutines.
func (r *Runner) Wait() { r.wg.Wait() }

// loop runs a single collector forever until ctx is done. An initial
// Run kicks off immediately so /api/health returns useful data on the
// first poll after startup instead of waiting one whole interval.
func (r *Runner) loop(ctx context.Context, c Collector) {
	defer r.wg.Done()

	log := r.logger.With("collector", c.Name(), "tier", c.Tier().String())

	// Eager first run so handlers see real data ASAP.
	if err := c.Run(ctx); err != nil {
		log.Warn("collector failed", "err", err)
	}

	// Context may have been cancelled during the first Run — bail early
	// so we don't start the ticker only to shut it down on the next tick.
	if ctx.Err() != nil {
		return
	}

	ticker := time.NewTicker(c.Tier().Interval())
	defer ticker.Stop()

	for {
		select {
		case <-ctx.Done():
			return
		case <-ticker.C:
			if err := c.Run(ctx); err != nil {
				log.Warn("collector failed", "err", err)
			}
		}
	}
}
```

```go
// into a runtime-queryable Topology. The file is written by the dashboard
// module's pkgs.writeText expression and passed via --config-file. A
// missing or empty path yields an empty Topology with no error — useful
// for dev runs on a Mac where no config file exists.
package topology

import (
	"encoding/json"
	"errors"
	"fmt"
	"io/fs"
	"os"
)

// Topology is the runtime mirror of the module-rendered config file.
type Topology struct {
	Tunnels          []Tunnel      `json:"tunnels"`
	Pools            []Pool        `json:"pools"`
	PooledRules      []PooledRule  `json:"pooled_rules"`
	StaticLeases     []StaticLease `json:"static_leases"`
	// AllowlistEnabled is true only when router.nftables.allowedMacs is
	// configured on this install. When false, the dashboard must treat
	// every client's allowlist status as n/a rather than "blocked" —
	// there is no policy in effect.
	AllowlistEnabled bool     `json:"allowlist_enabled"`
	AllowedMACs      []string `json:"allowed_macs"`
	LANInterface     string   `json:"lan_interface"`
}

type Tunnel struct {
	Name         string `json:"name"`
	Interface    string `json:"interface"`
	Fwmark       string `json:"fwmark"`
	RoutingTable int    `json:"routing_table"`
}

type Pool struct {
	Name    string   `json:"name"`
	Members []string `json:"members"`
}

type PooledRule struct {
	Sources []string `json:"sources"`
	Pool    string   `json:"pool"`
}

type StaticLease struct {
	MAC  string `json:"mac"`
	IP   string `json:"ip"`
	Name string `json:"name,omitempty"`
}

// Load reads the topology file at path. An empty path or a missing file
// returns an empty Topology with a nil error — call sites should log this
// at warn level. A present but malformed file returns a hard error.
func Load(path string) (*Topology, error) {
	if path == "" {
		return &Topology{}, nil
	}
	b, err := os.ReadFile(path)
	if err != nil {
		if errors.Is(err, fs.ErrNotExist) {
			return &Topology{}, nil
		}
		return nil, fmt.Errorf("read topology %s: %w", path, err)
	}
	var t Topology
	if err := json.Unmarshal(b, &t); err != nil {
		return nil, fmt.Errorf("parse topology %s: %w", path, err)
	}
	return &t, nil
}

// TunnelByName returns a pointer to the tunnel with the given name, or
// nil if no such tunnel exists.
func (t *Topology) TunnelByName(name string) *Tunnel {
	for i := range t.Tunnels {
		if t.Tunnels[i].Name == name {
			return &t.Tunnels[i]
		}
	}
	return nil
}

// PoolByName returns a pointer to the pool with the given name, or nil.
func (t *Topology) PoolByName(name string) *Pool {
	for i := range t.Pools {
		if t.Pools[i].Name == name {
			return &t.Pools[i]
		}
	}
	return nil
}

// ClientIPsForPool returns the union of source IPs from every PooledRule
// that references the named pool.
func (t *Topology) ClientIPsForPool(name string) []string {
	seen := map[string]struct{}{}
	var out []string
	for _, r := range t.PooledRules {
		if r.Pool != name {
			continue
		}
		for _, ip := range r.Sources {
			if _, ok := seen[ip]; ok {
				continue
			}
			seen[ip] = struct{}{}
			out = append(out, ip)
		}
	}
	return out
}
```

```go
package proc

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

// NetDevStats is one row of /proc/net/dev (counters, never rates).
type NetDevStats struct {
	Interface string
	RXBytes   uint64
	RXPackets uint64
	TXBytes   uint64
	TXPackets uint64
}

// ReadNetDev parses /proc/net/dev (or a fixture path) and returns one
// NetDevStats per interface keyed by interface name.
func ReadNetDev(path string) (map[string]NetDevStats, error) {
	f, err := os.Open(path)
	if err != nil {
		return nil, fmt.Errorf("open %s: %w", path, err)
	}
	defer f.Close()

	out := map[string]NetDevStats{}
	scanner := bufio.NewScanner(f)
	// Skip the 2-line header.
	for i := 0; i < 2 && scanner.Scan(); i++ {
	}
	for scanner.Scan() {
		line := scanner.Text()
		colon := strings.IndexByte(line, ':')
		if colon < 0 {
			continue
		}
		name := strings.TrimSpace(line[:colon])
		rest := strings.Fields(line[colon+1:])
		// /proc/net/dev columns after the colon:
		//  0:rx_bytes  1:rx_packets  2:rx_errs  3:rx_drop  4:rx_fifo
		//  5:rx_frame  6:rx_compressed  7:rx_multicast
		//  8:tx_bytes  9:tx_packets  10:tx_errs  11:tx_drop
		//  12:tx_fifo  13:tx_colls  14:tx_carrier  15:tx_compressed
		if len(rest) < 16 {
			continue
		}
		rxBytes, _ := strconv.ParseUint(rest[0], 10, 64)
		rxPkts, _ := strconv.ParseUint(rest[1], 10, 64)
		txBytes, _ := strconv.ParseUint(rest[8], 10, 64)
		txPkts, _ := strconv.ParseUint(rest[9], 10, 64)
		out[name] = NetDevStats{
			Interface: name,
			RXBytes:   rxBytes,
			RXPackets: rxPkts,
			TXBytes:   txBytes,
			TXPackets: txPkts,
		}
	}
	return out, scanner.Err()
}
```

```go
package proc

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

// CPUTimes are the jiffies counters from /proc/stat's aggregate `cpu` line.
type CPUTimes struct {
	User   uint64
	Nice   uint64
	System uint64
	Idle   uint64
	IOWait uint64
}

// Total returns the sum of every tracked counter.
func (c CPUTimes) Total() uint64 {
	return c.User + c.Nice + c.System + c.Idle + c.IOWait
}

// Delta returns this-minus-prev for each counter. Callers should treat
// this as "what happened in the interval between the two samples".
func (c CPUTimes) Delta(prev CPUTimes) CPUTimes {
	return CPUTimes{
		User:   c.User - prev.User,
		Nice:   c.Nice - prev.Nice,
		System: c.System - prev.System,
		Idle:   c.Idle - prev.Idle,
		IOWait: c.IOWait - prev.IOWait,
	}
}

// Stat is a subset of /proc/stat: aggregate CPU times plus boot time.
type Stat struct {
	CPUTimes
	BootTimeUnix int64
}

// ReadStat parses /proc/stat.
func ReadStat(path string) (Stat, error) {
	f, err := os.Open(path)
	if err != nil {
		return Stat{}, fmt.Errorf("open %s: %w", path, err)
	}
	defer f.Close()

	var s Stat
	scanner := bufio.NewScanner(f)
	for scanner.Scan() {
		line := scanner.Text()
		switch {
		case strings.HasPrefix(line, "cpu "):
			// "cpu  user nice system idle iowait irq softirq steal guest guest_nice"
			fields := strings.Fields(line)
			if len(fields) < 6 {
				continue
			}
			s.User, _ = strconv.ParseUint(fields[1], 10, 64)
			s.Nice, _ = strconv.ParseUint(fields[2], 10, 64)
			s.System, _ = strconv.ParseUint(fields[3], 10, 64)
			s.Idle, _ = strconv.ParseUint(fields[4], 10, 64)
			s.IOWait, _ = strconv.ParseUint(fields[5], 10, 64)
		case strings.HasPrefix(line, "btime "):
			fields := strings.Fields(line)
			if len(fields) == 2 {
				s.BootTimeUnix, _ = strconv.ParseInt(fields[1], 10, 64)
			}
		}
	}
	return s, scanner.Err()
}
```

```go
// Meminfo is the subset of /proc/meminfo the dashboard needs.
type Meminfo struct {
	TotalBytes     uint64
	AvailableBytes uint64
	FreeBytes      uint64
}

// ReadMeminfo parses /proc/meminfo. kB values are converted to bytes so
// downstream consumers don't need to remember the units.
func ReadMeminfo(path string) (Meminfo, error) {
	f, err := os.Open(path)
	if err != nil {
		return Meminfo{}, fmt.Errorf("open %s: %w", path, err)
	}
	defer f.Close()

	var m Meminfo
	scanner := bufio.NewScanner(f)
	for scanner.Scan() {
		line := scanner.Text()
		colon := strings.IndexByte(line, ':')
		if colon < 0 {
			continue
		}
		key := line[:colon]
		rest := strings.Fields(line[colon+1:])
		if len(rest) == 0 {
			continue
		}
		v, err := strconv.ParseUint(rest[0], 10, 64)
		if err != nil {
			continue
		}
		// /proc/meminfo values are in kB unless the suffix says otherwise;
		// all the fields we care about are kB.
		bytes := v * 1024
		switch key {
		case "MemTotal":
			m.TotalBytes = bytes
		case "MemAvailable":
			m.AvailableBytes = bytes
		case "MemFree":
			m.FreeBytes = bytes
		}
	}
	return m, scanner.Err()
}
```

```go
// ReadThermal reads /sys/class/thermal/thermal_zone0/temp, which is an
// integer in milli-degrees-Celsius (e.g. 52314 → 52.314 °C).
func ReadThermal(path string) (float64, error) {

}

	if err != nil {
		return 0, fmt.Errorf("parse uptime %q: %w", fields[0], err)
	}
	return v, nil
```

```go
package collector

import (
	"context"
	"fmt"
	"time"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/proc"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
)

const sampleSlots = 60

// TrafficOpts configures a Traffic collector.
type TrafficOpts struct {
	// NetDevPath is the path to /proc/net/dev. Injected for tests.
	NetDevPath string
	// Interfaces is the allowlist of interface names to track. Other
	// entries in /proc/net/dev are ignored (lo, docker bridges, etc.).
	Interfaces []string
	State      *state.State
}

// Traffic is the hot-tier collector that populates state.Traffic from
// /proc/net/dev. It keeps a per-interface ring buffer of 60 recent
// samples so the frontend can render sparklines without an extra query.
type Traffic struct {
	opts     TrafficOpts
	lastRead map[string]proc.NetDevStats
	lastTS   time.Time
	rings    map[string][]model.InterfaceSample
}

// NewTraffic constructs a Traffic collector.
func NewTraffic(opts TrafficOpts) *Traffic {
	return &Traffic{
		opts:     opts,
		lastRead: map[string]proc.NetDevStats{},
		rings:    map[string][]model.InterfaceSample{},
	}
}

func (c *Traffic) Name() string { return "traffic" }
func (c *Traffic) Tier() Tier   { return Hot }

// Run reads /proc/net/dev, computes deltas against the previous read,
// appends a sample to each interface's ring buffer, and writes the
// result into state.State.
func (c *Traffic) Run(_ context.Context) error {
	now := time.Now()
	stats, err := proc.ReadNetDev(c.opts.NetDevPath)
	if err != nil {
		return fmt.Errorf("read netdev: %w", err)
	}

	interfaces := make([]model.Interface, 0, len(c.opts.Interfaces))
	for _, name := range c.opts.Interfaces {
		cur, ok := stats[name]
		if !ok {
			// Interface went away (unlikely for eth0/eth1/wg_*). Skip.
			continue
		}

		var rxBps, txBps uint64
		if !c.lastTS.IsZero() {
			prev, hadPrev := c.lastRead[name]
			if hadPrev {
				elapsed := now.Sub(c.lastTS).Seconds()
				if elapsed > 0 {
					// Counters can wrap on 32-bit interfaces; guard against
					// negative delta by clamping. On aarch64 Linux they're
					// 64-bit, so this is a defence-in-depth.
					if cur.RXBytes >= prev.RXBytes {
						rxBps = uint64(float64(cur.RXBytes-prev.RXBytes) / elapsed * 8.0)
					}
					if cur.TXBytes >= prev.TXBytes {
						txBps = uint64(float64(cur.TXBytes-prev.TXBytes) / elapsed * 8.0)
					}
				}
			}
		}

		ring := c.rings[name]
		if len(ring) != sampleSlots {
			ring = make([]model.InterfaceSample, sampleSlots)
		}
		copy(ring, ring[1:])
		ring[sampleSlots-1] = model.InterfaceSample{RXBps: rxBps, TXBps: txBps}
		c.rings[name] = ring

		interfaces = append(interfaces, model.Interface{
			Name:         name,
			RXBps:        rxBps,
			TXBps:        txBps,
			RXBytesTotal: cur.RXBytes,
			TXBytesTotal: cur.TXBytes,
			Samples60s:   append([]model.InterfaceSample(nil), ring...),
		})
	}

	c.lastRead = stats
	c.lastTS = now
	c.opts.State.SetTraffic(model.Traffic{Interfaces: interfaces})
	return nil
}
```

```go
package collector

import (
	"context"
	"fmt"
	"time"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/proc"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
)

// SystemOpts configures the hot-tier System collector. Medium-tier
// fields (Services, Throttled) are populated by a separate collector;
// see Task 7.
type SystemOpts struct {
	StatPath    string
	MeminfoPath string
	ThermalPath string
	UptimePath  string
	State       *state.State
}

// System is the hot-tier collector for CPU / memory / temperature /
// uptime. It tracks the previous /proc/stat counters so it can compute
// percent-over-delta between runs.
type System struct {
	opts   SystemOpts
	lastCPU *proc.CPUTimes
}

func NewSystem(opts SystemOpts) *System { return &System{opts: opts} }

func (c *System) Name() string { return "system" }
func (c *System) Tier() Tier   { return Hot }

func (c *System) Run(_ context.Context) error {
	st, err := proc.ReadStat(c.opts.StatPath)
	if err != nil {
		return fmt.Errorf("read stat: %w", err)
	}
	mem, err := proc.ReadMeminfo(c.opts.MeminfoPath)
	if err != nil {
		return fmt.Errorf("read meminfo: %w", err)
	}
	temp, err := proc.ReadThermal(c.opts.ThermalPath)
	if err != nil {
		return fmt.Errorf("read thermal: %w", err)
	}
	upt, err := proc.ReadUptime(c.opts.UptimePath)
	if err != nil {
		return fmt.Errorf("read uptime: %w", err)
	}

	// Merge with whatever the medium-tier system collector already set
	// (services, throttled) so we don't clobber their fields.
	prev, _ := c.opts.State.SnapshotSystem()

	cur := prev
	cur.Memory = model.MemoryStats{
		TotalBytes:     mem.TotalBytes,
		AvailableBytes: mem.AvailableBytes,
		UsedBytes:      mem.TotalBytes - mem.AvailableBytes,
	}
	if mem.TotalBytes > 0 {
		cur.Memory.PercentUsed = float64(cur.Memory.UsedBytes) / float64(mem.TotalBytes) * 100.0
	}
	cur.TemperatureC = temp
	cur.UptimeSeconds = upt
	cur.BootTime = time.Unix(st.BootTimeUnix, 0).UTC()

	if c.lastCPU != nil {
		d := st.CPUTimes.Delta(*c.lastCPU)
		total := d.Total()
		if total > 0 {
			cur.CPU = model.CPUStats{
				PercentUser:   float64(d.User) / float64(total) * 100.0,
				PercentSystem: float64(d.System) / float64(total) * 100.0,
				PercentIdle:   float64(d.Idle) / float64(total) * 100.0,
				PercentIOWait: float64(d.IOWait) / float64(total) * 100.0,
			}
		}
	}
	// Remember this reading for the next delta.
	prev := st.CPUTimes
	c.lastCPU = &prev

	c.opts.State.SetSystem(cur)
	return nil
}
```

Note: we deliberately shadow `prev` inside the function — the outer `prev` holds a snapshot of State, while the inner one holds the just-read CPU counters. Rename for clarity:

```go

	lastCopy := st.CPUTimes
	c.lastCPU = &lastCopy
```

```go
// Package server assembles the dashboard HTTP handlers into a single mux.
package server

import (
	"encoding/json"
	"net/http"
	"runtime/debug"
	"time"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/config"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/spa"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/topology"
)

// Version is set at build time via -ldflags='-X .../internal/server.Version=...'.
var Version = ""

var (
	startedAt      = time.Now().UTC()
	runningVersion = readVersion()
)

func readVersion() string {
	if Version != "" {
		return Version
	}
	info, ok := debug.ReadBuildInfo()
	if !ok {
		return "dev"
	}
	if v := info.Main.Version; v != "" && v != "(devel)" {
		return v
	}
	var rev, modified string
	for _, s := range info.Settings {
		switch s.Key {
		case "vcs.revision":
			rev = s.Value
		case "vcs.modified":
			modified = s.Value
		}
	}
	if rev != "" {
		short := rev
		if len(short) > 7 {
			short = short[:7]
		}
		if modified == "true" {
			short += "-dirty"
		}
		return short
	}
	return "dev"
}

// New returns the top-level http.Handler with all routes wired.
//
// Route precedence (Go 1.22 ServeMux):
//
//  1. GET /api/...        — specific endpoints
//  2. /api                — exact path, JSON 404
//  3. /api/               — subtree catch-all, JSON 404
//  4. /                   — SPA fallback
//
// `st` holds collected data; `topo` holds eval-time router config. Both
// are read-only from the handler's perspective and fully thread-safe.
func New(_ *config.Config, st *state.State, topo *topology.Topology) http.Handler {
	mux := http.NewServeMux()

	mux.HandleFunc("GET /api/health", handleHealth)
	mux.HandleFunc("GET /api/traffic", handleTraffic(st))
	mux.HandleFunc("GET /api/system", handleSystem(st))

	mux.HandleFunc("/api", handleAPINotFound)
	mux.HandleFunc("/api/", handleAPINotFound)
	mux.Handle("/", spa.Handler())

	return mux
}

type healthResponse struct {
	OK        bool      `json:"ok"`
	Version   string    `json:"version"`
	StartedAt time.Time `json:"started_at"`
}

func handleHealth(w http.ResponseWriter, _ *http.Request) {
	writeJSON(w, http.StatusOK, healthResponse{
		OK:        true,
		Version:   runningVersion,
		StartedAt: startedAt,
	})
}

type apiError struct {
	Error string `json:"error"`
}

func handleAPINotFound(w http.ResponseWriter, r *http.Request) {
	writeJSON(w, http.StatusNotFound, apiError{
		Error: "no such api endpoint: " + r.URL.Path,
	})
}

func writeJSON(w http.ResponseWriter, code int, body any) {
	w.Header().Set("Content-Type", "application/json; charset=utf-8")
	w.WriteHeader(code)
	_ = json.NewEncoder(w).Encode(body)
}
```

```go
// Package systemd wraps `systemctl is-active` for the dashboard's service
// health card. The bulk-invocation form (`systemctl is-active u1 u2 u3`)
// produces one status line per unit, in order, so we parse it positionally.
package systemd

import (
	"context"
	"fmt"
	"os/exec"
	"strings"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
)

// Collect runs `systemctl is-active <units...>` and returns a ServiceState
// per unit. A non-zero exit from systemctl is expected when at least one
// unit is inactive/failed — the stdout still contains the per-unit answers,
// so we treat non-zero exit as "some unit isn't active" and still parse.
func Collect(ctx context.Context, units []string) ([]model.ServiceState, error) {
	if len(units) == 0 {
		return nil, nil
	}
	args := append([]string{"is-active"}, units...)
	cmd := exec.CommandContext(ctx, "systemctl", args...)
	out, err := cmd.Output()
	// Exit status 3 = "at least one unit is not active" — still a good run.
	if err != nil {
		if exitErr, ok := err.(*exec.ExitError); ok && exitErr.ExitCode() == 3 {
			// parse anyway
		} else if exitErr, ok := err.(*exec.ExitError); ok && len(exitErr.Stderr) > 0 {
			return nil, fmt.Errorf("systemctl is-active: %s", strings.TrimSpace(string(exitErr.Stderr)))
		} else if len(out) == 0 {
			return nil, fmt.Errorf("systemctl is-active: %w", err)
		}
	}
	return parseIsActive(units, string(out)), nil
}

// parseIsActive matches each line of stdout to the corresponding unit
// in the input slice. The ordering is guaranteed by systemd.
func parseIsActive(units []string, stdout string) []model.ServiceState {
	lines := strings.Split(strings.TrimRight(stdout, "\n"), "\n")
	out := make([]model.ServiceState, 0, len(units))
	for i, u := range units {
		state := "unknown"
		if i < len(lines) {
			state = strings.TrimSpace(lines[i])
		}
		out = append(out, model.ServiceState{
			Name:     u,
			Active:   state == "active",
			RawState: state,
		})
	}
	return out
}
```

```go
// Package vcgencmd wraps `vcgencmd get_throttled` for RPi thermal/voltage
// status. The raw output is "throttled=0xNNNNN" where any non-zero value
// indicates past or present throttling. We return both the raw string
// (for the UI to display) and a boolean "any bit set" flag.
package vcgencmd

import (
	"context"
	"fmt"
	"os/exec"
	"strings"
)

// Collect runs vcgencmd and returns (raw, flag, err).
func Collect(ctx context.Context) (string, bool, error) {
	cmd := exec.CommandContext(ctx, "vcgencmd", "get_throttled")
	out, err := cmd.Output()
	if err != nil {
		return "", false, fmt.Errorf("vcgencmd get_throttled: %w", err)
	}
	raw, flag := parseThrottled(strings.TrimSpace(string(out)))
	return raw, flag, nil
}

func parseThrottled(line string) (string, bool) {
	i := strings.IndexByte(line, '=')
	if i < 0 {
		return line, false
	}
	val := strings.TrimSpace(line[i+1:])
	return val, val != "" && val != "0x0"
}
```

```go
// Package wireguard parses `wg show <iface> dump` output into typed data.
// The command itself is invoked from the Tunnels collector — this package
// only deals with the string parsing so it is fully unit-testable.
package wireguard

import (
	"context"
	"fmt"
	"os/exec"
	"strconv"
	"strings"
)

// Dump is one interface's full `wg show <iface> dump` output.
type Dump struct {
	LocalPrivateKey string
	LocalPublicKey  string
	ListenPort      int
	Fwmark          string
	Peers           []Peer
}

// Peer is one remote endpoint row.
type Peer struct {
	PublicKey           string
	PresharedKey        string
	Endpoint            string
	AllowedIPs          string
	LatestHandshakeUnix int64
	RXBytes             uint64
	TXBytes             uint64
	PersistentKeepalive int
}

// ParseDump parses the stdout of `wg show <iface> dump`.
func ParseDump(raw string) (Dump, error) {
	lines := strings.Split(strings.TrimSpace(raw), "\n")
	if len(lines) == 0 || lines[0] == "" {
		return Dump{}, fmt.Errorf("wireguard: empty dump")
	}
	var d Dump

	head := strings.Split(lines[0], "\t")
	if len(head) < 3 {
		return Dump{}, fmt.Errorf("wireguard: short header %q", lines[0])
	}
	d.LocalPrivateKey = head[0]
	d.LocalPublicKey = head[1]
	if v, err := strconv.Atoi(head[2]); err == nil {
		d.ListenPort = v
	}
	if len(head) >= 4 {
		d.Fwmark = head[3]
	}

	for _, line := range lines[1:] {
		if line == "" {
			continue
		}
		f := strings.Split(line, "\t")
		if len(f) < 8 {
			continue
		}
		p := Peer{
			PublicKey:    f[0],
			PresharedKey: f[1],
			Endpoint:     f[2],
			AllowedIPs:   f[3],
		}
		p.LatestHandshakeUnix, _ = strconv.ParseInt(f[4], 10, 64)
		p.RXBytes, _ = strconv.ParseUint(f[5], 10, 64)
		p.TXBytes, _ = strconv.ParseUint(f[6], 10, 64)
		p.PersistentKeepalive, _ = strconv.Atoi(f[7])
		d.Peers = append(d.Peers, p)
	}
	return d, nil
}

// Show runs `wg show <iface> dump` via exec.CommandContext and parses the
// output. Errors from the command are wrapped with context.
func Show(ctx context.Context, iface string) (Dump, error) {
	cmd := exec.CommandContext(ctx, "wg", "show", iface, "dump")
	out, err := cmd.Output()
	if err != nil {
		if exitErr, ok := err.(*exec.ExitError); ok && len(exitErr.Stderr) > 0 {
			return Dump{}, fmt.Errorf("wg show %s: %s", iface, strings.TrimSpace(string(exitErr.Stderr)))
		}
		return Dump{}, fmt.Errorf("wg show %s: %w", iface, err)
	}
	return ParseDump(string(out))
}
```

```go
// Package poolhealth parses /run/wg-pool-health/state.json — a JSON file
// maintained by the router's wg-pool-health.service systemd unit. The
// file is atomic-rewritten every ~5 s with the current health of each
// WireGuard tunnel the pool-health watchdog probes.
package poolhealth

import (
	"encoding/json"
	"errors"
	"fmt"
	"io/fs"
	"os"
)

// State mirrors the on-disk schema written by wg-pool-health.service:
//   { "updated_at": "...", "tunnels": { "<name>": {"healthy": bool, "consecutive_failures": int} } }
type State struct {
	UpdatedAt string                `json:"updated_at"`
	Tunnels   map[string]TunnelInfo `json:"tunnels"`
}

// TunnelInfo is one tunnel's health record.
type TunnelInfo struct {
	Healthy              bool `json:"healthy"`
	ConsecutiveFailures  int  `json:"consecutive_failures"`
}

// ReadState reads and parses the state file. A missing file yields an
// empty State with no error — that happens at every boot before
// wg-pool-health writes its first state file, and briefly during config
// reloads. Hard-erroring would turn that transient into a warning log on
// every collector tick.
func ReadState(path string) (State, error) {
	b, err := os.ReadFile(path)
	if err != nil {
		if errors.Is(err, fs.ErrNotExist) {
			return State{Tunnels: map[string]TunnelInfo{}}, nil
		}
		return State{}, fmt.Errorf("read %s: %w", path, err)
	}
	var s State
	if err := json.Unmarshal(b, &s); err != nil {
		return State{}, fmt.Errorf("parse %s: %w", path, err)
	}
	if s.Tunnels == nil {
		s.Tunnels = map[string]TunnelInfo{}
	}
	return s, nil
}
```

```go
package collector

import (
	"context"
	"time"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/poolhealth"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/wireguard"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/topology"
)

// ShowFunc matches wireguard.Show. Callers can inject a fake for tests.
type ShowFunc func(ctx context.Context, iface string) (wireguard.Dump, error)

// TunnelsOpts configures the Tunnels collector.
type TunnelsOpts struct {
	Topology       *topology.Topology
	PoolHealthPath string
	State          *state.State
	Show           ShowFunc // nil → wireguard.Show
}

// Tunnels is the hot-tier collector that combines `wg show <iface> dump`
// with /run/wg-pool-health/state.json for every tunnel in topology.
type Tunnels struct {
	opts TunnelsOpts
}

// NewTunnels constructs a Tunnels collector.
func NewTunnels(opts TunnelsOpts) *Tunnels {
	if opts.Show == nil {
		opts.Show = wireguard.Show
	}
	return &Tunnels{opts: opts}
}

func (c *Tunnels) Name() string { return "tunnels" }
func (c *Tunnels) Tier() Tier   { return Hot }

func (c *Tunnels) Run(ctx context.Context) error {
	// Pool-health file read is cheap — do it once and index by name.
	health, _ := poolhealth.ReadState(c.opts.PoolHealthPath)

	out := make([]model.Tunnel, 0, len(c.opts.Topology.Tunnels))
	for _, t := range c.opts.Topology.Tunnels {
		m := model.Tunnel{
			Name:         t.Name,
			Interface:    t.Interface,
			Fwmark:       t.Fwmark,
			RoutingTable: t.RoutingTable,
		}

		dump, err := c.opts.Show(ctx, t.Interface)
		if err == nil && len(dump.Peers) > 0 {
			p := dump.Peers[0]
			m.PublicKey = p.PublicKey
			m.Endpoint = p.Endpoint
			m.RXBytes = p.RXBytes
			m.TXBytes = p.TXBytes
			if p.LatestHandshakeUnix > 0 {
				m.LatestHandshakeSecondsAgo = time.Now().Unix() - p.LatestHandshakeUnix
			} else {
				m.LatestHandshakeSecondsAgo = -1
			}
		}

		if hinfo, ok := health.Tunnels[t.Name]; ok {
			m.Healthy = hinfo.Healthy
			m.ConsecutiveFailures = hinfo.ConsecutiveFailures
		} else {
			// No pool-health entry: assume healthy (the tunnel isn't pooled).
			m.Healthy = true
		}

		out = append(out, m)
	}
	c.opts.State.SetTunnels(out)
	return nil
}
```

```go
package collector

import (
	"context"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/poolhealth"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/topology"
)

// PoolsOpts configures the Pools collector.
type PoolsOpts struct {
	Topology       *topology.Topology
	PoolHealthPath string
	State          *state.State
}

// Pools is the hot-tier collector that exposes pool membership with
// per-member health. Flow counts stay at 0 — a cold-tier companion
// (Task 12) populates them from conntrack.
type Pools struct {
	opts PoolsOpts
}

func NewPools(opts PoolsOpts) *Pools { return &Pools{opts: opts} }

func (c *Pools) Name() string { return "pools" }
func (c *Pools) Tier() Tier   { return Hot }

func (c *Pools) Run(_ context.Context) error {
	health, _ := poolhealth.ReadState(c.opts.PoolHealthPath)

	// Index tunnel meta by name for fwmark lookup.
	fwmarks := map[string]string{}
	for _, t := range c.opts.Topology.Tunnels {
		fwmarks[t.Name] = t.Fwmark
	}

	// Preserve previously-collected per-member flow counts so the cold
	// tier's contribution survives hot-tier rewrites.
	prev, _ := c.opts.State.SnapshotPools()
	prevFlows := map[string]map[string]int{}
	for _, p := range prev {
		m := map[string]int{}
		for _, mem := range p.Members {
			m[mem.Tunnel] = mem.FlowCount
		}
		prevFlows[p.Name] = m
	}

	out := make([]model.Pool, 0, len(c.opts.Topology.Pools))
	for _, tp := range c.opts.Topology.Pools {
		pool := model.Pool{
			Name:      tp.Name,
			ClientIPs: c.opts.Topology.ClientIPsForPool(tp.Name),
		}
		allUnhealthy := len(tp.Members) > 0
		for _, name := range tp.Members {
			mem := model.PoolMember{
				Tunnel: name,
				Fwmark: fwmarks[name],
			}
			if h, ok := health.Tunnels[name]; ok {
				mem.Healthy = h.Healthy
			} else {
				mem.Healthy = true
			}
			if mem.Healthy {
				allUnhealthy = false
			}
			if flows, ok := prevFlows[tp.Name]; ok {
				mem.FlowCount = flows[name]
			}
			pool.Members = append(pool.Members, mem)
		}
		pool.FailsafeDropActive = allUnhealthy
		out = append(out, pool)
	}
	c.opts.State.SetPools(out)
	return nil
}
```

```go
// Package conntrack wraps `conntrack-tools` for cold-tier flow
// aggregation. We never invoke conntrack from a request handler;
// collectors run it on the 30 s tier.
package conntrack

import (
	"context"
	"fmt"
	"os/exec"
	"strings"
)

// Runner executes a conntrack command and returns its stdout. Injected
// for tests; production runs `exec.CommandContext`.
type Runner func(ctx context.Context, args ...string) (string, error)

// DefaultRunner exec-invokes the real conntrack binary.
func DefaultRunner(ctx context.Context, args ...string) (string, error) {
	cmd := exec.CommandContext(ctx, "conntrack", args...)
	out, err := cmd.Output()
	if err != nil {
		if exitErr, ok := err.(*exec.ExitError); ok && len(exitErr.Stderr) > 0 {
			return "", fmt.Errorf("conntrack %v: %s", args, strings.TrimSpace(string(exitErr.Stderr)))
		}
		return "", fmt.Errorf("conntrack %v: %w", args, err)
	}
	return string(out), nil
}

// CountByFwmark counts active flows whose conntrack mark matches fwmark.
// Output format per line is the default conntrack -L plain-text flow row.
func CountByFwmark(ctx context.Context, run Runner, fwmark string) (int, error) {
	out, err := run(ctx, "-L", "-m", fwmark)
	if err != nil {
		return 0, err
	}
	count := 0
	for _, line := range strings.Split(out, "\n") {
		if strings.TrimSpace(line) == "" {
			continue
		}
		count++
	}
	return count, nil
}
```

```go
import (
	"context"
	"fmt"
	"strings"
)

// ClientFwmarks runs `conntrack -L` and returns a map of "first seen"
// client-IP → fwmark for flows with a non-zero mark. Flows with mark=0
// (WAN path) are excluded. When multiple flows from the same source IP
// have different marks (rare but possible with mixed pools), the first
// observed mark wins — good enough for the dashboard's coarse
// "current tunnel" hint.
func ClientFwmarks(ctx context.Context, run Runner) (map[string]string, error) {
	out, err := run(ctx, "-L")
	if err != nil {
		return nil, err
	}
	result := map[string]string{}
	for _, line := range strings.Split(out, "\n") {
		if strings.TrimSpace(line) == "" {
			continue
		}
		src := extractField(line, "src=")
		mark := extractField(line, "mark=")
		if src == "" || mark == "" || mark == "0" {
			continue
		}
		if _, already := result[src]; already {
			continue
		}
		n := 0
		_, _ = fmt.Sscanf(mark, "%d", &n)
		if n == 0 {
			continue
		}
		result[src] = fmt.Sprintf("0x%x", n)
	}
	return result, nil
}

// extractField returns the value of `key` in a whitespace-separated
// "k=v" soup. Returns "" if not present.
func extractField(line, key string) string {
	i := strings.Index(line, key)
	if i < 0 {
		return ""
	}
	rest := line[i+len(key):]
	end := strings.IndexAny(rest, " \t")
	if end < 0 {
		return rest
	}
	return rest[:end]
}
```

```go
// PoolFlows is the cold-tier (30 s) companion to Pools. It runs
// `conntrack -L -m <fwmark>` once per distinct tunnel fwmark and merges
// the counts into the existing pools state, preserving everything else.
type PoolFlows struct {
	opts PoolFlowsOpts
}

// NewPoolFlows constructs a PoolFlows collector.
func NewPoolFlows(opts PoolFlowsOpts) *PoolFlows {
	if opts.Run == nil {
		opts.Run = conntrack.DefaultRunner
	}
	return &PoolFlows{opts: opts}
}

func (c *PoolFlows) Name() string { return "pool-flows" }
func (c *PoolFlows) Tier() Tier   { return Cold }

func (c *PoolFlows) Run(ctx context.Context) error {
	// Collect the set of distinct fwmarks actually used by pools.
	marks := map[string]string{} // fwmark → tunnel name
	for _, t := range c.opts.Topology.Tunnels {
		marks[t.Fwmark] = t.Name
	}

	counts := map[string]int{}
	for fwmark, name := range marks {
		n, err := conntrack.CountByFwmark(ctx, c.opts.Run, fwmark)
		if err != nil {
			return err
		}
		counts[name] = n
	}

	// Merge into existing pool state without racing the hot-tier Pools
	// collector: read, mutate, write.
	pools, _ := c.opts.State.SnapshotPools()
	for i := range pools {
		for j := range pools[i].Members {
			pools[i].Members[j].FlowCount = counts[pools[i].Members[j].Tunnel]
		}
	}
	c.opts.State.SetPools(pools)
	return nil
}
```

```go
// Package dnsmasq parses the dnsmasq.leases file format:
//
//	<expire_unix> <mac> <ip> <hostname_or_star> <client_id_or_star>
//
// Fields are whitespace-separated, one lease per line. A missing file
// is treated as "no leases" rather than an error.
package dnsmasq

import (
	"bufio"
	"errors"
	"fmt"
	"io/fs"
	"os"
	"strconv"
	"strings"
	"time"
)

// Lease is one row from dnsmasq.leases.
type Lease struct {
	ExpireUnix int64
	MAC        string
	IP         string
	Hostname   string // "" when dnsmasq logs "*"
	ClientID   string // "" when dnsmasq logs "*"
}

// ExpiresAt returns ExpireUnix as a time.Time in UTC. A zero ExpireUnix
// stays zero.
func (l Lease) ExpiresAt() time.Time {
	if l.ExpireUnix == 0 {
		return time.Time{}
	}
	return time.Unix(l.ExpireUnix, 0).UTC()
}

// ReadLeases parses the dnsmasq.leases file at path.
func ReadLeases(path string) ([]Lease, error) {
	f, err := os.Open(path)
	if err != nil {
		if errors.Is(err, fs.ErrNotExist) {
			return nil, nil
		}
		return nil, fmt.Errorf("open %s: %w", path, err)
	}
	defer f.Close()

	var out []Lease
	scanner := bufio.NewScanner(f)
	for scanner.Scan() {
		fields := strings.Fields(scanner.Text())
		if len(fields) < 4 {
			continue
		}
		exp, _ := strconv.ParseInt(fields[0], 10, 64)
		l := Lease{
			ExpireUnix: exp,
			MAC:        strings.ToLower(fields[1]),
			IP:         fields[2],
		}
		if fields[3] != "*" {
			l.Hostname = fields[3]
		}
		if len(fields) >= 5 && fields[4] != "*" {
			l.ClientID = fields[4]
		}
		out = append(out, l)
	}
	return out, scanner.Err()
}
```

```go
// Package ipneigh wraps `ip neigh show` for MAC discovery. The output is
// line-oriented; we return a map of IP → MAC for non-FAILED entries.
package ipneigh

import (
	"context"
	"fmt"
	"os/exec"
	"strings"
)

// Collect runs `ip neigh show` and returns a map of IPv4 → MAC. FAILED
// entries are filtered out.
func Collect(ctx context.Context) (map[string]string, error) {
	cmd := exec.CommandContext(ctx, "ip", "neigh", "show")
	out, err := cmd.Output()
	if err != nil {
		if exitErr, ok := err.(*exec.ExitError); ok && len(exitErr.Stderr) > 0 {
			return nil, fmt.Errorf("ip neigh: %s", strings.TrimSpace(string(exitErr.Stderr)))
		}
		return nil, fmt.Errorf("ip neigh: %w", err)
	}
	return parseNeigh(string(out)), nil
}

// parseNeigh is pulled out for unit testing.
func parseNeigh(raw string) map[string]string {
	out := map[string]string{}
	for _, line := range strings.Split(raw, "\n") {
		line = strings.TrimSpace(line)
		if line == "" {
			continue
		}
		fields := strings.Fields(line)
		if len(fields) < 2 {
			continue
		}
		ip := fields[0]
		// Find "lladdr <mac>" pair.
		var mac, state string
		for i := 0; i < len(fields); i++ {
			switch fields[i] {
			case "lladdr":
				if i+1 < len(fields) {
					mac = strings.ToLower(fields[i+1])
				}
			case "REACHABLE", "STALE", "DELAY", "PROBE", "PERMANENT", "NOARP", "INCOMPLETE", "FAILED":
				state = fields[i]
			}
		}
		if mac == "" || state == "FAILED" || state == "INCOMPLETE" {
			continue
		}
		out[ip] = mac
	}
	return out
}
```

```go
package collector

import (
	"context"
	"sort"
	"strings"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/model"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/sources/dnsmasq"

	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/state"
	"github.com/giko/nixos-rpi4-router/modules/dashboard/backend/internal/topology"
)

// NeighFunc matches ipneigh.Collect; injected for tests.
type NeighFunc func(ctx context.Context) (map[string]string, error)

// ClientsOpts configures the Clients collector.
type ClientsOpts struct {
	Topology   *topology.Topology
	LeasesPath string
	State      *state.State
	NeighFunc  NeighFunc // nil → ipneigh.Collect
}

// Clients is the medium-tier collector for LAN hosts. It merges three
// sources (topology static leases, dnsmasq dynamic leases, ip neigh) and
// overlays current-tunnel from the cold-tier client-fwmark cache.
type Clients struct {
	opts ClientsOpts
}

// NewClients constructs a Clients collector.
func NewClients(opts ClientsOpts) *Clients {
	if opts.NeighFunc == nil {
		opts.NeighFunc = ipneigh.Collect
	}
	return &Clients{opts: opts}
}

func (c *Clients) Name() string { return "clients" }
func (c *Clients) Tier() Tier   { return Medium }

func (c *Clients) Run(ctx context.Context) error {
	leases, _ := dnsmasq.ReadLeases(c.opts.LeasesPath)
	neigh, _ := c.opts.NeighFunc(ctx)
	fwmarks, _ := c.opts.State.SnapshotClientFwmarks()

	topo := c.opts.Topology
	allowed := map[string]struct{}{}
	if topo.AllowlistEnabled {
		for _, m := range topo.AllowedMACs {
			allowed[strings.ToLower(m)] = struct{}{}
		}
		// Static-leased MACs are implicitly allowed (see nftables module spec).
		for _, sl := range topo.StaticLeases {
			allowed[strings.ToLower(sl.MAC)] = struct{}{}
		}
	}

	// Index tunnel by fwmark for the current-tunnel lookup.
	tunnelByMark := map[string]string{}
	for _, t := range topo.Tunnels {
		tunnelByMark[t.Fwmark] = t.Name
	}

	// Figure out, for each source IP, which pool it's part of.
	poolOfIP := map[string]string{}
	for _, pr := range topo.PooledRules {
		for _, ip := range pr.Sources {
			poolOfIP[ip] = pr.Pool
		}
	}

	merged := map[string]model.Client{}

	// 1) Static leases from topology.
	for _, sl := range topo.StaticLeases {
		merged[sl.IP] = model.Client{
			Hostname:  sl.Name,
			IP:        sl.IP,
			MAC:       strings.ToLower(sl.MAC),
			LeaseType: "static",
		}
	}

	// 2) dnsmasq leases — only fill in gaps; don't override static ones.
	for _, l := range leases {
		if _, already := merged[l.IP]; already {
			continue
		}
		merged[l.IP] = model.Client{
			Hostname:  l.Hostname,
			IP:        l.IP,
			MAC:       l.MAC,
			LeaseType: "dynamic",
			LastSeen:  l.ExpiresAt(),
		}
	}

	// 3) ip neigh — for devices with no lease (IoT with static IPs, etc.).
	for ip, mac := range neigh {
		if _, already := merged[ip]; already {
			// Update MAC if missing (rare, but possible).
			cl := merged[ip]
			if cl.MAC == "" {
				cl.MAC = mac
				merged[ip] = cl
			}
			continue
		}
		merged[ip] = model.Client{
			IP:        ip,
			MAC:       mac,
			LeaseType: "neighbor",
		}
	}

	// 4) Overlay route + current_tunnel + allowlist.
	out := make([]model.Client, 0, len(merged))
	for _, cl := range merged {
		if mark, ok := fwmarks[cl.IP]; ok {
			if t, ok := tunnelByMark[mark]; ok {
				cl.CurrentTunnel = t
			}
		}
		if pool, ok := poolOfIP[cl.IP]; ok {
			cl.Route = "pool:" + pool
		} else if cl.CurrentTunnel != "" {
			cl.Route = "tunnel:" + cl.CurrentTunnel
		} else {
			cl.Route = "wan"
		}
		switch {
		case !topo.AllowlistEnabled:
			// No MAC allowlist is in effect on this install — there is no
			// "allowed vs blocked" axis to show. Labelling everyone blocked
			// would be factually wrong (nothing is being dropped) and
			// labelling everyone allowed would imply a policy that does
			// not exist. n/a is the honest answer.
			cl.AllowlistStatus = "n/a"
		case cl.MAC == "":
			cl.AllowlistStatus = "n/a"
		default:
			if _, ok := allowed[cl.MAC]; ok {
				cl.AllowlistStatus = "allowed"
			} else {
				cl.AllowlistStatus = "blocked"
			}
		}
		out = append(out, cl)
	}

	// Sort by IP for stable ordering across refreshes.
	sort.Slice(out, func(i, j int) bool { return out[i].IP < out[j].IP })

	c.opts.State.SetClients(out)
	return nil
}
```

```go
// Package adguard wraps AdGuard Home's REST API (/control/*). The
// dashboard polls /control/stats on the medium tier; /control/querylog
// is proxied on demand from /api/adguard/querylog.
package adguard

import (
	"context"
	"encoding/json"
	"fmt"
	"io"
	"net/http"
	"net/url"
	"strings"
	"time"
)

// Client is a thin HTTP wrapper around AdGuard Home's REST API.
type Client struct {
	base *url.URL
	http *http.Client
}

// NewClient constructs a Client. Pass http.DefaultClient (or an injected
// httptest client) as the second arg. The returned Client fails hard on
// construction if base can't be parsed — that's a config error.
func NewClient(base string, httpClient *http.Client) *Client {
	u, err := url.Parse(base)
	if err != nil {
		// Falling back to an empty URL means every call errors out
		// gracefully rather than panicking at startup.
		u = &url.URL{}
	}
	if httpClient == nil {
		httpClient = &http.Client{Timeout: 5 * time.Second}
	}
	return &Client{base: u, http: httpClient}
}

// Stats is the subset of /control/stats the dashboard uses. AdGuard's
// schema is larger; we ignore fields we don't need.
type Stats struct {
	NumDNSQueries int         `json:"num_dns_queries"`
	NumBlocked    int         `json:"num_blocked_filtering"`
	TopBlocked    []TopDomain // flattened from top_blocked_domains
	TopQueried    []TopDomain // flattened from top_queried_domains
	TopClients    []TopClient
	// 24 per-hour bins. The API returns separate dns_queries and
	// blocked_filtering slices of length `stats_interval` in hours; we
	// map them 1:1 into DensityBin.
	Density []DensityBin
}

type TopDomain struct {
	Domain string
	Count  int
}

type TopClient struct {
	IP    string
	Count int
}

type DensityBin struct {
	StartHour int
	Queries   int
	Blocked   int
}

// rawStats mirrors the literal AdGuard JSON schema. AdGuard's "top *"
// arrays are [{"domain": count}, ...] — a slice of single-key maps, which
// we normalise below.
type rawStats struct {
	NumDNSQueries     int                `json:"num_dns_queries"`
	NumBlocked        int                `json:"num_blocked_filtering"`
	TopQueried        []map[string]int   `json:"top_queried_domains"`
	TopBlocked        []map[string]int   `json:"top_blocked_domains"`
	TopClients        []map[string]int   `json:"top_clients"`
	DNSQueries        []int              `json:"dns_queries"`
	BlockedFiltering  []int              `json:"blocked_filtering"`
}

// FetchStats calls GET <base>/control/stats and decodes the response.
func (c *Client) FetchStats(ctx context.Context) (Stats, error) {
	body, err := c.get(ctx, "/control/stats")
	if err != nil {
		return Stats{}, err
	}
	var raw rawStats
	if err := json.Unmarshal(body, &raw); err != nil {
		return Stats{}, fmt.Errorf("adguard /control/stats: %w", err)
	}
	return normalise(raw), nil
}

func normalise(raw rawStats) Stats {
	flatten := func(in []map[string]int) []TopDomain {
		out := make([]TopDomain, 0, len(in))
		for _, m := range in {
			for k, v := range m {
				out = append(out, TopDomain{Domain: k, Count: v})
			}
		}
		return out
	}
	flattenClients := func(in []map[string]int) []TopClient {
		out := make([]TopClient, 0, len(in))
		for _, m := range in {
			for k, v := range m {
				out = append(out, TopClient{IP: k, Count: v})
			}
		}
		return out
	}
	density := make([]DensityBin, 0, 24)
	for i := 0; i < len(raw.DNSQueries) && i < 24; i++ {
		bin := DensityBin{StartHour: i, Queries: raw.DNSQueries[i]}
		if i < len(raw.BlockedFiltering) {
			bin.Blocked = raw.BlockedFiltering[i]
		}
		density = append(density, bin)
	}
	return Stats{
		NumDNSQueries: raw.NumDNSQueries,
		NumBlocked:    raw.NumBlocked,
		TopQueried:    flatten(raw.TopQueried),
		TopBlocked:    flatten(raw.TopBlocked),
		TopClients:    flattenClients(raw.TopClients),
		Density:       density,
	}
}

// QueryLogOptions constrain a /control/querylog fetch.
type QueryLogOptions struct {
	Limit  int
	Client string
	Domain string
}

// QueryLogResponse is the raw shape the dashboard's own querylog endpoint
// hands back to the frontend. AdGuard's /control/querylog returns a larger
// object keyed by "data" — we pass that array through verbatim so the
// frontend can render whatever fields exist.
type QueryLogResponse struct {
	Raw json.RawMessage
}

// FetchQueryLog calls /control/querylog and returns the raw JSON body
// so the querylog handler can forward it with its own caching envelope.
// FetchQueryLog calls /control/querylog and returns the normalised
// queries array as JSON. AdGuard's upstream response is {"data": [...]}
// but the dashboard's spec contract is {"queries": [...]}, so we
// extract the "data" key here so the handler can wrap it verbatim.
func (c *Client) FetchQueryLog(ctx context.Context, opts QueryLogOptions) (json.RawMessage, error) {
	q := url.Values{}
	if opts.Limit > 0 {
		q.Set("limit", fmt.Sprintf("%d", opts.Limit))
	}
	// AdGuard's /control/querylog has a single `search` parameter that
	// matches against client IP, domain name, and upstream. When the
	// dashboard caller passes both `client` and `domain`, we concatenate
	// them with a space — AdGuard treats the whole string as one text
	// match, which narrows the results. Not perfect (it's substring, not
	// AND), but better than silently dropping one filter.
	switch {
	case opts.Client != "" && opts.Domain != "":
		q.Set("search", opts.Client+" "+opts.Domain)
	case opts.Client != "":
		q.Set("search", opts.Client)
	case opts.Domain != "":
		q.Set("search", opts.Domain)
	}
	path := "/control/querylog"
	if len(q) > 0 {
		path += "?" + q.Encode()
	}
	body, err := c.get(ctx, path)
	if err != nil {
		return nil, err
	}
	// Normalise: AdGuard returns {"data": [...], ...} — extract the
	// "data" array so callers get a bare JSON array of query entries
	// that the handler wraps in {"queries": ...} per spec §7.4.
	var upstream struct {
		Data json.RawMessage `json:"data"`
	}
	if err := json.Unmarshal(body, &upstream); err != nil {
		return body, nil
	}
	if upstream.Data == nil {
		return []byte("[]"), nil
	}
	return upstream.Data, nil
}

func (c *Client) get(ctx context.Context, path string) (json.RawMessage, error) {
	if c.base == nil || c.base.Host == "" {
		return nil, fmt.Errorf("adguard: no base url")
	}
	u := *c.base
	// Trim trailing slash so path concatenation is clean.
	u.Path = strings.TrimRight(u.Path, "/") + path
	// AdGuard HMAC-free localhost API; GET.
	req, err := http.NewRequestWithContext(ctx, "GET", u.String(), nil)
	if err != nil {
		return nil, err
	}
	resp, err := c.http.Do(req)
	if err != nil {
		return nil, fmt.Errorf("adguard GET %s: %w", u.Path, err)
	}
	defer resp.Body.Close()
	if resp.StatusCode != 200 {
		return nil, fmt.Errorf("adguard GET %s: status %d", u.Path, resp.StatusCode)
	}
	return io.ReadAll(resp.Body)
}
```

```go
// queryLogCache coalesces identical AdGuard querylog fetches.
// Responses live for 3 s keyed by the full incoming query string, which
// is safe because that's the natural granularity of a TanStack-Query
// filter refresh.
type queryLogCache struct {
	client *adguard.Client
	group  singleflight.Group

	mu    sync.Mutex
	cache map[string]queryLogEntry
}

type queryLogEntry struct {
	body      json.RawMessage
	expiresAt time.Time
}

const queryLogTTL = 3 * time.Second

func newQueryLogCache(client *adguard.Client) *queryLogCache {
	return &queryLogCache{
		client: client,
		cache:  map[string]queryLogEntry{},
	}
}

// fetch returns a (possibly cached) /control/querylog response.
func (q *queryLogCache) fetch(r *http.Request) (json.RawMessage, error) {
	key := r.URL.RawQuery

	q.mu.Lock()
	if ent, ok := q.cache[key]; ok && time.Now().Before(ent.expiresAt) {
		q.mu.Unlock()
		return ent.body, nil
	}
	q.mu.Unlock()

	// Build options outside the singleflight closure so all concurrent
	// callers contribute their query string, but the fetch itself runs
	// on a detached context. Using r.Context() inside Do would tie the
	// shared fetch to the first caller — if that tab disconnects, every
	// waiting request fails together.
	opts := adguard.QueryLogOptions{Limit: 200}
	if l := r.URL.Query().Get("limit"); l != "" {
		opts.Limit = parseIntDefault(l, 200)
	}
	opts.Client = r.URL.Query().Get("client")
	opts.Domain = r.URL.Query().Get("domain")

	result, err, _ := q.group.Do(key, func() (any, error) {
		ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
		defer cancel()

		raw, err := q.client.FetchQueryLog(ctx, opts)
		if err != nil {
			return nil, err
		}
		q.mu.Lock()
		q.cache[key] = queryLogEntry{
			body:      raw,
			expiresAt: time.Now().Add(queryLogTTL),
		}
		for k, v := range q.cache {
			if time.Now().After(v.expiresAt) {
				delete(q.cache, k)
			}
		}
		q.mu.Unlock()
		return raw, nil
	})
	if err != nil {
		return nil, err
	}
	return result.(json.RawMessage), nil
}

func parseIntDefault(s string, def int) int {
	var v int
	if _, err := fmtSscanf(s, "%d", &v); err == nil && v > 0 {
		return v
	}
	return def
}

// fmtSscanf is a tiny shim so this file doesn't import fmt twice via the
// err path; kept lowercase to avoid polluting package exports.
var fmtSscanf = func(s, format string, a ...any) (int, error) {
	return innerSscanf(s, format, a...)
}

func handleAdguardQueryLog(cache *queryLogCache) http.HandlerFunc {
	return func(w http.ResponseWriter, r *http.Request) {
		raw, err := cache.fetch(r)
		if err != nil {
			envelope.WriteJSON(w, http.StatusBadGateway, map[string]any{
				"error": err.Error(),
			}, time.Now().UTC(), true)
			return
		}
		// `raw` is already the normalised array of query entries (the
		// AdGuard client extracts "data" from the upstream response).
		// Wrap it under "queries" per spec §7.4.
		envelope.WriteJSON(w, http.StatusOK, map[string]json.RawMessage{
			"queries": raw,
		}, time.Now().UTC(), false)
	}
}
```

```go
// Package ratelimit is defence-in-depth on top of the nftables allowlist.
// The primary trust boundary is still the firewall (spec §9); this
// middleware contains bugs in *allowed* clients (runaway browser tabs,
// reconnect loops) from saturating the collector.
package ratelimit

import (
	"encoding/json"
	"net"
	"net/http"
	"sync"

	"golang.org/x/time/rate"
)

// Options configures the middleware.
type Options struct {
	PerIPPerSec  float64 // tokens per second per source IP
	PerIPBurst   int
	GlobalPerSec float64 // global tokens per second
	GlobalBurst  int
}

// Default returns the values from spec §7.6.
func Default() Options {
	return Options{
		PerIPPerSec:  20,
		PerIPBurst:   40,
		GlobalPerSec: 200,
		GlobalBurst:  400,
	}
}

// New returns a middleware that applies both buckets to every request
// EXCEPT /api/health, which is intentionally exempt so monitoring probes
// don't burn bucket tokens.
func New(opts Options) func(http.Handler) http.Handler {
	global := rate.NewLimiter(rate.Limit(opts.GlobalPerSec), opts.GlobalBurst)

	var mu sync.Mutex
	perIP := map[string]*rate.Limiter{}

	getLimiter := func(ip string) *rate.Limiter {
		mu.Lock()
		defer mu.Unlock()
		if l, ok := perIP[ip]; ok {
			return l
		}
		l := rate.NewLimiter(rate.Limit(opts.PerIPPerSec), opts.PerIPBurst)
		perIP[ip] = l
		return l
	}

	return func(next http.Handler) http.Handler {
		return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
			if r.URL.Path == "/api/health" {
				next.ServeHTTP(w, r)
				return
			}
			// Check per-IP BEFORE global. If a single noisy client is
			// over its per-IP budget we reject immediately without
			// consuming a global token. The previous order (global first)
			// let one runaway tab burn through the shared 200 rps budget
			// and start 429-ing unrelated admin clients.
			ip, _, err := net.SplitHostPort(r.RemoteAddr)
			if err != nil {
				ip = r.RemoteAddr
			}
			if !getLimiter(ip).Allow() {
				tooMany(w)
				return
			}
			if !global.Allow() {
				tooMany(w)
				return
			}
			next.ServeHTTP(w, r)
		})
	}
}

func tooMany(w http.ResponseWriter) {
	w.Header().Set("Content-Type", "application/json; charset=utf-8")
	w.Header().Set("Retry-After", "1")
	w.WriteHeader(http.StatusTooManyRequests)
	_ = json.NewEncoder(w).Encode(map[string]any{
		"error":              "rate limit exceeded",
		"retry_after_seconds": 1,
	})
}
```

```ts
/**
 * Display formatters. All input values are in SI / kernel units (bits per
 * second, bytes, seconds, Unix ms, etc.) — formatters decide the unit.
 */

const BPS_UNITS = ["bps", "Kbps", "Mbps", "Gbps", "Tbps"];
const BYTE_UNITS = ["B", "KiB", "MiB", "GiB", "TiB"];

/**
 * formatBps — "1.4 Mbps". Chooses the nearest kilo-boundary so values
 * settle on a stable unit while scrolling (no unit jitter between
 * samples).
 */
export function formatBps(bps: number): string {
  if (bps < 1) return "0 bps";
  let v = bps;
  let i = 0;
  while (v >= 1000 && i < BPS_UNITS.length - 1) {
    v /= 1000;
    i++;
  }
  return `${v.toFixed(v < 10 ? 1 : 0)} ${BPS_UNITS[i]}`;
}

/**
 * formatBytes — "42.5 MiB". Binary unit prefixes because counter fields
 * are powers-of-two in kernel semantics.
 */
export function formatBytes(bytes: number): string {
  if (bytes < 1) return "0 B";
  let v = bytes;
  let i = 0;
  while (v >= 1024 && i < BYTE_UNITS.length - 1) {
    v /= 1024;
    i++;
  }
  return `${v.toFixed(v < 10 ? 1 : 0)} ${BYTE_UNITS[i]}`;
}

/**
 * formatDuration — "14d 2h" / "3h 42m" / "17s". Aimed at uptime displays;
 * never returns more than two non-zero units.
 */
export function formatDuration(seconds: number): string {
  if (seconds < 1) return "0s";
  const s = Math.floor(seconds);
  const d = Math.floor(s / 86400);
  const h = Math.floor((s % 86400) / 3600);
  const m = Math.floor((s % 3600) / 60);
  const rem = s % 60;
  if (d > 0) return `${d}d ${h}h`;
  if (h > 0) return `${h}h ${m}m`;
  if (m > 0) return `${m}m ${rem}s`;
  return `${rem}s`;
}

/**
 * formatPercent — "34%" from 0..1 fraction or 0..100 number. We detect
 * fractional form by whether the input is ≤ 1.
 */
export function formatPercent(n: number): string {
  const pct = n <= 1 ? n * 100 : n;
  return `${pct.toFixed(0)}%`;
}

/**
 * formatRelativeAgo — "12m 4s ago", "just now", "stale".
 */
export function formatRelativeAgo(iso: string | null | undefined): string {
  if (!iso) return "never";
  const t = new Date(iso).getTime();
  if (Number.isNaN(t)) return "—";
  const diffMs = Date.now() - t;
  if (diffMs < 1500) return "just now";
  const s = Math.floor(diffMs / 1000);
  if (s < 60) return `${s}s ago`;
  const m = Math.floor(s / 60);
  const rem = s % 60;
  if (m < 60) return `${m}m ${rem}s ago`;
  const h = Math.floor(m / 60);
  return `${h}h ${m % 60}m ago`;
}

/**
 * formatAbsoluteTime — "14:22:04" (local time). Used on log rows.
 */
export function formatAbsoluteTime(iso: string | null | undefined): string {
  if (!iso) return "—";
  const d = new Date(iso);
```

```tsx
import { ReactNode, useMemo, useState } from "react";
import { cn } from "@/lib/utils";

export type Column<Row> = {
  key: string;
  header: ReactNode;
  /** Return a cell node given the row. */
  cell: (row: Row) => ReactNode;
  /** Optional: return a value used for sorting instead of the rendered cell. */
  sortValue?: (row: Row) => string | number;
  className?: string;
  align?: "left" | "right" | "center";
};

/**
 * DataTable is a minimal sortable table. It purposefully does NOT wrap
 * shadcn's `<Table>` so that the header cell typography matches the
 * Silent Sentinel designs (label-xs, muted). Click a header to sort
 * ascending; click again to toggle.
 */
export function DataTable<Row>({
  columns,
  rows,
  rowKey,
  onRowClick,
  emptyMessage = "No rows.",
  className,
}: {
  columns: Column<Row>[];
  rows: Row[];
  rowKey: (row: Row) => string;
  onRowClick?: (row: Row) => void;
  emptyMessage?: string;
  className?: string;
}) {
  const [sort, setSort] = useState<{ key: string; dir: "asc" | "desc" } | null>(null);

  const sorted = useMemo(() => {
    if (!sort) return rows;
    const col = columns.find((c) => c.key === sort.key);
    if (!col || !col.sortValue) return rows;
    const cp = [...rows];
    cp.sort((a, b) => {
      const va = col.sortValue!(a);
      const vb = col.sortValue!(b);
      if (va < vb) return sort.dir === "asc" ? -1 : 1;
      if (va > vb) return sort.dir === "asc" ? 1 : -1;
      return 0;
    });
    return cp;
  }, [rows, sort, columns]);

  return (
    <div className={cn("bg-surface-lowest rounded-sm overflow-hidden", className)}>
      <table className="w-full text-left border-collapse">
        <thead>
          <tr className="bg-surface-low">
            {columns.map((c) => (
              <th
                key={c.key}
                className={cn(
                  "px-4 py-3 text-[10px] uppercase tracking-widest font-black text-on-surface-variant",
                  c.align === "right" && "text-right",
                  c.align === "center" && "text-center",
                  c.sortValue && "cursor-pointer select-none",
                  c.className,
                )}
                onClick={() => {
                  if (!c.sortValue) return;
                  setSort((s) =>
                    s && s.key === c.key
                      ? { key: c.key, dir: s.dir === "asc" ? "desc" : "asc" }
                      : { key: c.key, dir: "asc" },
                  );
                }}
              >
                {c.header}
                {sort?.key === c.key && (
                  <span className="ml-1 opacity-60">{sort.dir === "asc" ? "↑" : "↓"}</span>
                )}
              </th>
            ))}
          </tr>
        </thead>
        <tbody>
          {sorted.length === 0 && (
            <tr>
              <td colSpan={columns.length} className="px-4 py-8 text-center text-on-surface-variant text-xs">
                {emptyMessage}
              </td>
            </tr>
          )}
          {sorted.map((row) => (
            <tr
              key={rowKey(row)}
              className={cn(
                "hover:bg-surface-low/40 transition-colors",
                onRowClick && "cursor-pointer",
              )}
              onClick={() => onRowClick?.(row)}
            >
              {columns.map((c) => (
                <td
                  key={c.key}
                  className={cn(
                    "px-4 py-3 text-sm",
                    c.align === "right" && "text-right",
                    c.align === "center" && "text-center",
                    c.className,
                  )}
                >
                  {c.cell(row)}
                </td>
              ))}
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}
```

```tsx
import { useMemo, useState } from "react";
import { useQuery } from "@tanstack/react-query";
import { Layout } from "@/components/Layout";
import { DataTable } from "@/components/DataTable";
import { MonoText } from "@/components/MonoText";
import { StatusBadge } from "@/components/StatusBadge";
import { ClientBadge } from "@/components/ClientBadge";
import { PoolBadge } from "@/components/PoolBadge";
import { StaleIndicator } from "@/components/StaleIndicator";
import { api, type Client } from "@/lib/api";
import { queryKeys } from "@/lib/query-keys";
import { cn } from "@/lib/utils";

type Filter = "all" | "vpn" | "wan" | "static" | "unknown";

const FILTERS: { key: Filter; label: string }[] = [
  { key: "all", label: "All Devices" },
  { key: "vpn", label: "On VPN" },
  { key: "wan", label: "Direct to WAN" },
  { key: "static", label: "Static Leases" },
  { key: "unknown", label: "Unknown" },
];

// "Unknown" matches devices that don't have a stable identity on this
// LAN — no hostname, neighbour-discovered (not leased), or carrying an
// allowlist_status of "blocked" (actively rejected by the MAC allowlist).
// This mirrors the "Unknown" filter pill in designs/02-clients.html and
// lets the owner quickly spot anything unexpected on the network.
function matchesFilter(c: Client, filter: Filter): boolean {
  switch (filter) {
    case "all":
      return true;
    case "vpn":
      return c.route.startsWith("pool:") || c.route.startsWith("tunnel:");
    case "wan":
      return c.route === "wan";
    case "static":
      return c.lease_type === "static";
    case "unknown":
      return c.lease_type === "neighbor" || c.hostname === "" || c.allowlist_status === "blocked";
  }
}

export function Clients() {
  const clients = useQuery({
    queryKey: queryKeys.clients(),
    queryFn: api.clients,
    refetchInterval: 5_000,
  });

  const [filter, setFilter] = useState<Filter>("all");
  const [search, setSearch] = useState("");

  const rows = useMemo(() => {
    const list = clients.data?.data.clients ?? [];
    const s = search.toLowerCase();
    return list
      .filter((c) => matchesFilter(c, filter))
      .filter((c) =>
        s === ""
          ? true
          : c.hostname.toLowerCase().includes(s) ||
            c.ip.includes(s) ||
            c.mac.includes(s),
      );
  }, [clients.data, filter, search]);

  return (
    <Layout title="Clients">
      <div className="flex items-center justify-between mt-6 mb-4">
        <h1 className="text-xl font-semibold">
          Connected Clients
          <span className="ml-3 align-middle">
            <StatusBadge kind="info">{rows.length} SHOWN</StatusBadge>
          </span>
        </h1>
        <StaleIndicator stale={clients.data?.stale ?? true} updatedAt={clients.data?.updated_at ?? null} />
      </div>

      <div className="flex flex-wrap items-center justify-between gap-4 mb-4">
        <div className="flex items-center gap-2 overflow-x-auto">
          {FILTERS.map((f) => (
            <button
              key={f.key}
              type="button"
              onClick={() => setFilter(f.key)}
```

## Rollback plan

If the router-dashboard binary segfaults or misbehaves on the Pi after deploy:

1. `nixos-rebuild switch --rollback` on the Pi (reverts to the previous generation — includes the previous dashboard binary).
2. Alternatively, pin `modules/dashboard/version.json` back to the previous `version` / `url` / `hash` by hand and push a `git revert` commit.
3. The backend's systemd unit is `Restart=on-failure`, so a transient panic won't block other services — the nftables allowlist rule for port 9090 stays in place even if the dashboard is down.
